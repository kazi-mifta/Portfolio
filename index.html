Measure Feet

Java | Android OS | Android Studio | Java Graphics | Image Processing

Measure Feet is an Android App to Measure several Parameteres of your Feet. This app was designed specially for the Diabetes Patients. Diabetes Patient suffer a problem Where Their Feet Shape Changes almost Every month. So They Need To get Shoes that are customized for their Feet. The Shoes are made from Cloth. This app will assist them to measure their Feet parameters. Length, Width, Arc Height of Human Feet can be Measured using this App.

  • Date: August 2018
  • Code: Private




Livy TV App

Java | Android OS | Android Studio | Android TV Leanback Library

Livy is an application that allows a user to download any type of File to Their Amazon TV or Any AndroidOS TV. It uses the Slim Downloader Library which can easily Pause, Resume any Download File as well as Manage multiple downloads at Once. The app can be easily operated using the Fire TV Remote.

  • Code: GitHub
