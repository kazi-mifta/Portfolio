Parking Space Detection

Python | Anaconda | Image Processing | Machine Learning

Parking Space Detection is a Python Project to Empty Space in a given Image. It is divided into 2 sections. In section 1 the Given image is proecessed without machine Learning. First the White Boundary is detected and Then divided into rectangles. Then we check inside the rectangles to see if they contain anything that is like a car. Many Image processing techniques are used here including Edge Detection, Color Detection, Contour Dectection, Noise reduction etc. The second part is based on Machine Learning completely. It uses the MobileNet Caffe Model to Detect Cars in a image and Output how many empty space is left.

  • Date: November 2018
  • Code: GitHub

Bangla Sentece Analyzer is a sentence type estimation software. It's built with python and a yaml databse. There is several templates for 5 types of senteces.
